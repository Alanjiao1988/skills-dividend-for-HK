# 中文提示词：红利股分析

请你扮演红利股研究员，基于公开资料分析以下股票：{股票代码}。

投资者假设：HK 税务居民个人，通过普通券商账户持有股票，目标是中长期现金分红收入。

请优先使用公司年报、中报、业绩公告、派息公告、交易所公告、公司 IR 材料和管理层公开表态。第三方行情数据只能作为交叉验证。

请输出：

1. 公司名称、代码、交易所、注册地、上市结构、报告货币、派息货币。
2. 当前股息率：区分普通股息和特别股息，计算税前和预估税后股息率。
3. 过去五年分红历史：DPS、普通股息、特别股息、派息率、现金流覆盖率、是否削减或暂停。
4. 管理层资本分配态度：分红政策、回购政策、降债、再投资和并购优先级。
5. 回购质量：是否真正减少股本，是否被股权激励稀释抵消，是否债务驱动。
6. 财务安全性：收入、利润、经营现金流、资本开支、自由现金流、净债务、利息覆盖和债务到期。
7. 未来三年分红预测：Bear / Base / Bull 三种情景。
8. 红利陷阱检查：高股息是否来自股价下跌、FCF 是否覆盖、杠杆是否上升、是否依赖资产出售或一次性收益。
9. 最终评级：分红质量、分红安全性、税后效率、回购质量、三年展望和组合角色。

请明确区分事实和判断，并列出来源。
