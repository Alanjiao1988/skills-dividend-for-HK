# Dividend Income Equity Analysis Skill

## 技能定位

本技能用于分析上市公司的现金分红价值，重点关注税前股息率、净股息率、逐年股息变化、逐年股息率、现金流覆盖、管理层资本分配态度、回购质量和未来三年分红跑道。

## 默认假设

- 投资者为 HK resident individual。
- 使用普通券商账户持有股票。
- 投资目标是中长期现金分红收入。
- 如果用户提供券商流水，券商实际扣税记录优先于理论分类。
- 本技能默认不处理内地个人港股通渠道，除非用户明确要求。

## 支撑文件读取规则

执行分析时按需读取同目录文件：

- `workflow.md`：完整研究流程、数据源、年度分红轨迹、现金流桥和红利陷阱检查。数据源优先级以该文件为准。
- `withholding-notes.md`：预扣税和券商实测扣税规则，是税务处理唯一权威源。
- `scoring.md`：100 分评分模型和评分锚点。
- `output-template.md`：最终输出模板，必须使用其中的直观表格结构。
- `schema.json`：仅当用户要求结构化输出、JSON 输出或机器可读结果时使用。
- `examples/example-output-skeleton.md`：输出样例骨架。

## 执行原则

- 报价、派息、财务数据必须注明 as-of date。
- 普通股息、特别股息、可变股息、REIT 分派和基金分派必须分开。
- 不得把一次性特别股息年化为持续收入。
- 必须用表格展示逐年 DPS、DPS YoY、当年股息率、按当前股价反推的历史股息率、派息率和现金流覆盖。
- 必须用三年情景表展示未来 DPS、净股息率、预计现金流、分红现金支出和覆盖倍数。
- 最终结论必须区分事实、假设和判断。

## 必需输出章节

1. 执行摘要。
2. 公司、上市地、注册地和证券结构。
3. 税务处理和券商实测扣税状态。
4. 当前税前和税后股息率。
5. 年度分红轨迹与逐年股息率。
6. 现金流覆盖和分红资金来源。
7. 管理层资本分配态度。
8. 回购质量。
9. 未来三年 Bear / Base / Bull 分红跑道。
10. 红利陷阱检查。
11. 评分和组合角色。
12. 数据来源和不确定性说明。
